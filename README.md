# Spring Boot Task 1

## Description
This is a simple web application. It displays a greeting message and the school logo.

## How to run
1. Clone the repository and open it in IntelliJ IDEA.
2. Run the `TeymurTask1Application` class.
3. Open the browser at: `http://localhost:8080/greeting?name=Vistula`.

## Features
* **Spring Web**: Used for the controller and HTTP requests.
* **Thymeleaf**: Template engine to show data in HTML.
* **Lombok**: To reduce boilerplate code.
* **Maven**: Project management and build tool.
